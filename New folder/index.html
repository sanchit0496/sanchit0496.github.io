Have you asked yourself, “What is Responsive Web design?”

Responsive Web design is an approach whereby a designer creates a Web page that “responds to” or resizes itself depending on the type of device it is being seen through. That could be an oversized desktop computer monitor, a laptop, a 10-inch tablet, a 7-inch tablet, or a 4-inch smartphone screen. Responsive Web design has become one of the hottest trends in 2013.
